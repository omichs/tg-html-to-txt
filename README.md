# 📨 tg-html-to-txt

Конвертер экспорта Telegram из HTML в чистый текст.

Telegram Desktop позволяет экспортировать историю переписки в виде HTML-файлов. Этот скрипт превращает их в читаемый `.txt` с правильными именами отправителей, датами и поддержкой многоядерных CPU.

---

## ✨ Возможности

- 🗂 Автоматически находит `messages.html`, `messages2.html`, ... в папке со скриптом
- 👤 Корректно определяет отправителя в **личных диалогах** (не путает «вас» и собеседника) и в **групповых чатах**
- 📅 Форматирует дату и день недели на русском языке
- 🚫 Пропускает сервисные уведомления Telegram
- ⚡ Многопроцессорный парсинг — задействует все доступные ядра CPU
- 💾 Два режима вывода: **один файл** (`full_history_converted.txt`) или **отдельный файл на каждый HTML**
- 🔤 Полная поддержка UTF-8 (без проблем с кодировкой на Windows)

## 🚀 Установка и запуск

### 1. Клонировать репозиторий

```bash
git clone https://github.com/omichs/tg-html-to-txt.git
cd tg-html-to-txt
```

### 2. Установить зависимости

```bash
pip install beautifulsoup4 tqdm
```

### 3. Скопировать скрипт в папку с экспортом Telegram

Положите `tg_html_to_txt.py` рядом с файлами `messages.html`, `messages2.html` и т.д.

```
📁 Ваша папка экспорта/
├── messages.html
├── messages2.html
├── messages3.html
└── tg_html_to_txt.py   ← скрипт сюда
```

### 4. Запустить

```bash
python tg_html_to_txt.py
```

Скрипт сам найдёт HTML-файлы рядом с собой и задаст два вопроса:
- Как сохранить результат (один файл или по отдельности)?
- Сколько процессов использовать для парсинга?

Если HTML-файлы лежат в другой папке, путь можно передать аргументом:

```bash
python tg_html_to_txt.py "C:\Users\Вы\Downloads\Telegram Export"
```

## 📄 Формат вывода

```
[15.03.2024 14:22 Пятница] Иван Петров: Привет!
[15.03.2024 14:23 Пятница] Вы: Привет, как дела?
[15.03.2024 14:23 Пятница] Иван Петров: Всё хорошо, спасибо.
    Кстати, ты видел новость про...
    Там такое написали, что я не поверил.
```

Многострочные сообщения: первая строка содержит заголовок, каждая следующая — с отступом в 4 пробела.

## ⚙️ Параметры запуска

| Параметр | Описание |
|---|---|
| *(без аргументов)* | Ищет HTML в папке скрипта, затем спрашивает путь |
| `python tg_html_to_txt.py <путь>` | Явно указать папку с экспортом |

Во время работы скрипт интерактивно спрашивает:

1. **Режим сохранения**
   - `1` — всё в один файл `full_history_converted.txt`
   - `2` — отдельный файл на каждый HTML (`messages00.txt`, `messages01.txt`, ...)

2. **Число процессов** — по умолчанию `min(кол-во ядер, 8)`, можно задать вручную
